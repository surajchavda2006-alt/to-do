About This To-Do App

This is a simple static To-Do web application built using HTML, CSS, JavaScript, and Bootstrap. It allows users to add, mark complete, and delete tasks to help organize daily activities.

The app stores your tasks locally in your browser, so your list stays even if you refresh the page.
